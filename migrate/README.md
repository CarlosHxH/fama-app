# migrate

To install dependencies:

```bash
bun install
```

To run:

```bash
bun run 
```

This project was created using `bun init` in bun v1.2.22. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
  Como usar

  # Migração completa
  npm run migrate

  # Testar sem gravar
  npm run migrate:dry

  # Só verificar se os dados batem
  npm run migrate:reconcile

  # Retomar do step jazigos (se customers/contratos já estão ok)
  npx tsx --env-file=.env migrate.ts --from=jazigos
